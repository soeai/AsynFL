## Prerequisites
- Python 3.9
- Docker

## Setting up the development environment
1. Run the following command to start MinIO and RabbitMQ
```
docker-compose up -d
```
2. Install dependencies 
```
pip3 install -r requirements.txt
```
or (if you use Windows)
```
pip install -r requirements.txt
```
